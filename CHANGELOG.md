# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-18

### Added
- Core agent automation engine built in Python 3.12.
- Multi-feed RSS parser supporting AWS What's New, Kubernetes Blog, CNCF Blog, and Docker Blog.
- Bedrock Nova Lite integration for content deduplication, filtering, summarization, and educational content generation.
- Responsive HTML email builder and Amazon SES integration.
- AWS SAM Infrastructure as Code (template.yaml) with Scheduler trigger and least-privilege IAM policies.
- Unit testing framework with mocks for RSS, Bedrock, and SES.
- Comprehensive documentation: Architecture, Deployment, Local Setup, Troubleshooting, Cost Estimation, and Roadmap.
