# Security Policy

## Supported Versions

We actively support and patch the latest release of OpsBeacon AI.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do not report it via public GitHub issues. Instead, please report it via email to security@opsbeacon.ai.

We will acknowledge receipt of your vulnerability report within 48 hours and work with you to investigate and resolve the issue. We follow a coordinated disclosure model.
